A few things that I found useful

Randomiser

This randomiser will give a random choice. Separate each choice with a comma (,)

Factor Finder

It will check if the number is a prime number, if its not the multiples will be listed

Random Number Generator

This will randomly generate a number between the given numbers. Seperate the minimum number and maximum number with a comma (,)

Solving quadratic equations

It will return a solution to the quadratic equation

x^2 + xy + y^2 = 0
